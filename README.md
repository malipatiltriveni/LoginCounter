## Description
A Java Servlet application that counts user logins using cookies.
Each login increments a counter stored as a cookie for 30 days.
Displays "This is your Nth login" on the welcome page.

## Technologies Used
- Java Servlets
- Apache Tomcat
- JSP
- HTML/CSS
- Cookies

2. Run on Apache Tomcat v9/v10
